# Vibe Coding Now — Project Overview

## Суть проекта

**Vibe Coding Now** — одностраничный showcase-сайт, демонстрирующий возможности современной веб-разработки с помощью нейросетей. Каждый блок — это живая демонстрация одной конкретной возможности: анимаций, интерактива, визуальных эффектов, адаптивности.

> «Создано за 1 день с помощью Claude и Cursor»

---

## Цели

| Цель | Описание |
|------|----------|
| Вдохновить | Показать, что современный веб — это не скучные формы, а живые интерфейсы |
| Обучить | Каждый блок — наглядный пример техники, которую можно изучить |
| Продвинуть | CTA в footer ведёт на курс по вайбкодингу |

---

## Целевая аудитория

- **Новички** — хотят понять, что возможно с нейросетями
- **Ученики курса** — видят результат, к которому идут
- **Опытные разработчики** — черпают вдохновение и новые техники

---

## Структура страницы (MVP)

```
1. Hero          — первое впечатление, анимированный заголовок + визуал
2. Animations    — stagger, scroll-trigger, Framer Motion showcase
3. Interactivity — конфетти, морфинг кнопок, слайдеры, переключатели
4. Visual FX     — glassmorphism, glow, градиенты, hover-эффекты
5. Responsive    — демо адаптивности с переключателем устройств
6. Footer        — инфо о проекте, GitHub, CTA
```

---

## Стиль и UX-принципы

- **Тёмная тема** как основа (с возможностью переключения на светлую)
- **Неоновые акценты**: кислотный зелёный, электрик-пурпур, огненный оранжевый
- **"Воздух"** между секциями — минимум 120px padding
- **Каждый блок** — отдельная визуальная история с уникальным фоном
- **Scroll-based** анимации: элементы появляются при прокрутке
- Никаких скучных статичных страниц — всё должно дышать

---

## Ключевые метрики успеха

- Первый экран загружается < 1.5s
- Lighthouse Performance > 85
- Все анимации работают без лагов (60fps)
- Сайт полностью работает на мобильных устройствах
